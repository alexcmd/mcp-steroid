# 0.93.0 Release Social Posts

## LinkedIn Post

IntelliJ for AI Agents, not humans -- MCP Steroid v0.93.0 is out.

Three changes that matter:
1. The project is full open under Apache 2.0. We are open for collaboration, contributions, and proof-of-concepts.

2. We rewrote what agents see when they connect. MCP Steroid now introduces itself as a stateful, drop-in replacement for LSP. Three-step flow: list projects → read the skill guide → execute code. The old routing table that agents ignored in 69 out of 69 arena runs is gone.

3. Fixed a bug where Maven test execution was killed mid-run. The modality monitor confused a coroutine progress entity with a UI dialog. Maven and Gradle integration tests now validate end-to-end in Docker.

4. Replaced 6 internal IntelliJ APIs with public alternatives. Down from 29 verifier warnings to 4. Forward compatibility for 2026.1 and beyond.

The real lesson from this release: we ran 87 agent sessions and measured what agents actually look at. Tool descriptions — yes. MCP server instructions — background context only. Resource guides — never read voluntarily. So we optimized for how agents work, not how we wish they worked.

The evals repo is open: github.com/jonnyzzz/mcp-steroid. 84+ prompt resources, Docker integration tests for Claude, Codex, and Gemini, DPAIA arena benchmarks. Contribute a scenario, run the evals, improve a skill.
